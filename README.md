# ⚡ DevClub IA - Gerador de Sites

Um projeto desenvolvido durante meus estudos de HTML, CSS e JavaScript, inspirado nas aulas do DevClub.

O objetivo é criar uma ferramenta capaz de gerar páginas web a partir da descrição de um negócio. Atualmente o projeto gera uma página de exemplo e exibe o código HTML e sua pré-visualização em tempo real.
